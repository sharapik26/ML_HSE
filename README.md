## Структура репозитория

### BonusHW
Бонусное задание: Named Entity Recognition (NER) для русских новостей о Brexit.

- Сравнение классических ML методов (sklearn) с LLM (DeepSeek)
- Модели: LogisticRegression, GaussianNB, LinearSVC, RandomForest
- Анализ bias-variance tradeoff, overfitting, generalization
- Визуализация результатов и анализ ошибок

**Файлы:**
- `BonusHW.ipynb` — Jupyter Notebook с решением
- `model_predictions.csv` — CSV с предсказаниями моделей
